# Rust Music Tracker

@README.md
@docs/VISION.md
@docs/roadmap.org

I want to make a music tracker really capable and really visually appealing, ergonomic with coding abilities like in strudel.
And Configurable, integrate well with external programs if needed.
Capable of creating any genre of music.
It should be capable enough to create complex music like dariacore, breakcore, or YTPMVs.

Currently, it's just a TUI app, but as things get complicated, I'm willing to make it a full GUI app.

I want the app to be as user-friendly as possible. If we can make the app intuitive somehow while still being feature-rich, we should aim for it. If it's realistically not possible, then we should at least hide advanced features from the user, but still allow them to access them in certain ways.
